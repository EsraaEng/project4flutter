# 📝To-Do App (Cubit State Management)

A simple and clean To-Do application built with Flutter using Cubit for state management.

##  Features

- Add new tasks
- Delete tasks
- Mark tasks as completed
- Dynamic task list
- Empty state message when there are no tasks

##  Tech Stack

- Flutter
- Dart
- Flutter Bloc (Cubit)

##  Bonus tasks
* [Mark task as complete]
* [Display a massage when there are no task]
